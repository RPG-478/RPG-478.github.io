プライバシーポリシー

最終更新日: 2026年2月11日

本プライバシーポリシーは、The Immovable Scroll（以下「本サービス」）におけるデータの取扱いについて説明します。

要約: 本サービスは個人情報を一切収集しません。すべてのデータはお使いのブラウザ内（localStorage）にのみ保存され、外部サーバーへの送信は行いません。

1. 収集する情報

本サービスが収集・保存する情報は以下のとおりです。すべてお使いのブラウザの localStorage にのみ保存されます。

データ項目用途保存場所
ハイスコア（px値）最高記録の表示localStorage
慣性モード設定操作モードの記憶localStorage
キャリブレーション値距離計測の補正localStorage

2. 収集しない情報

本サービスは以下の情報を一切収集しません。

  • 氏名、メールアドレス、電話番号などの個人情報
  • 位置情報
  • Cookie
  • アクセスログ・閲覧履歴
  • 端末識別子・広告ID

3. 外部サービスとの連携

本サービスは GitHub Pages（GitHub, Inc. 運営）上でホスティングされています。GitHub Pages のサーバーがアクセスログ（IPアドレス等）を記録する可能性がありますが、これは運営者の管理下にはありません。詳細は GitHub Pages のプライバシーに関するドキュメントをご参照ください。

本サービスは、上記以外の外部サービス（アクセス解析、広告配信、SNS連携等）を一切使用していません。

4. データの保存と削除

  • データはお使いのブラウザの localStorage にのみ保存されます。
  • 外部サーバーへのデータ送信は行いません。
  • データの削除は、ブラウザの設定からサイトデータを消去するか、アプリ内の「RESET」ボタンをご利用ください。

5. 子どものプライバシー

本サービスは個人情報を収集しないため、年齢制限を設けていません。

6. ポリシーの変更

本ポリシーは必要に応じて更新することがあります。変更があった場合は本ページに掲載します。

Privacy Policy

Last updated: February 11, 2026

This Privacy Policy describes how data is handled in The Immovable Scroll ("the Service").

Summary: The Service does not collect any personal information. All data is stored exclusively in your browser's localStorage and is never transmitted to external servers.

1. Information We Store

The following data is stored only in your browser's localStorage:

DataPurposeStorage
High score (px)Display personal bestlocalStorage
Inertia mode settingRemember control modelocalStorage
Calibration valueDistance measurement correctionlocalStorage

2. Information We Do NOT Collect

The Service does not collect:

  • Personal information (name, email, phone number, etc.)
  • Location data
  • Cookies
  • Access logs or browsing history
  • Device identifiers or advertising IDs

3. Third-Party Services

The Service is hosted on GitHub Pages (operated by GitHub, Inc.). GitHub Pages servers may record access logs (such as IP addresses), but this is outside the operator's control. See GitHub Pages privacy documentation for details.

No other third-party services (analytics, advertising, social media integrations, etc.) are used.

4. Data Storage and Deletion

  • Data is stored only in your browser's localStorage.
  • No data is transmitted to external servers.
  • To delete your data, clear your browser's site data or use the "RESET" button within the app.

5. Children's Privacy

Since the Service does not collect personal information, there are no age restrictions.

6. Changes to This Policy

This policy may be updated as needed. Changes will be posted on this page.
